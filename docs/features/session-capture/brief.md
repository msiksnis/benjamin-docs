# session-capture Brief

Session capture is the first wedge for `benjamin-docs`. A user should be able to work through an idea, product, feature, or handoff with an agent, then ask the agent to turn the conversation into durable project memory.

The output should be concise, structured Markdown plus metadata, not a raw transcript. It should help a future human, designer, developer, or agent understand what was decided and what remains uncertain.
